BTask is a website use to manage and also create To-Do-List with easy and simple UI

Uses Node React.JS, Chakra.UI and json-server


To run simply type this on console:
npm i @chakra-ui/react@2 @emotion/react @emotion/styled framer-motion
npm i @chakra-ui/icons

npm install -g json-server
json-server -w ./data/db.json

npm start
npm run dev

*do npm start and run dev if error shows
